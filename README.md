# ClipForge

Most of the best moments in a lecture, podcast, or workshop never get seen.
They are buried somewhere in the middle of a 60-minute video that nobody
has time to watch. ClipForge fixes that.

Upload a long-form video. ClipForge analyzes it, finds the moment that
hits hardest, crops it for vertical screens, and burns in timed captions —
all without you touching a single frame.

---

## The Problem

Educators and creators pour hours into recording sessions that most people
never finish watching. The insight that could change someone's perspective
sits at the 38-minute mark of a video with 200 views. It never reaches
the people it was meant for.

Short-form content is how people actually discover things in 2026. But
manually cutting, cropping, and captioning a single clip takes longer than
recording the original session. Most creators skip it entirely.

---

## What ClipForge Does

ClipForge runs a four-stage pipeline on any uploaded video:

**Peak Detection**
Librosa analyzes the audio energy across the entire video and identifies
the three moments where the speaker is most engaged. These are not random
cuts — they are the moments where something real is being said.

**Transcript Analysis**
OpenAI Whisper transcribes the full audio with word-level timestamps.
Google Gemini then reads the transcript alongside the detected peaks and
selects the single best 60-second clip — the one most likely to make
someone stop scrolling.

**Smart Vertical Crop**
OpenCV detects the speaker's face in the selected segment and crops the
horizontal video to 9:16, keeping the speaker centered throughout. No
black bars. No awkward framing.

**Caption Overlay**
Karaoke-style captions are burned directly onto the final video at the
correct timestamps. A hook headline generated by Gemini appears in the
opening seconds to capture attention immediately.

The output is a single MP4 file, ready to upload.

---

## Tech Stack

- Streamlit — application interface
- Google Gemini 2.0 Flash — clip selection and headline generation
- OpenAI Whisper — audio transcription with timestamps
- Librosa — audio energy analysis and peak detection
- MoviePy — video cutting, cropping, and caption rendering
- OpenCV — face detection for smart crop alignment
- Python — core pipeline and backend logic

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/Jayanth2319/ClipForge.git
cd ClipForge
pip install -r requirements.txt
```

Create a `.env` file in the root directory and add your Gemini API key:

```env
GEMINI_API_KEY="your_api_key_here"
```
